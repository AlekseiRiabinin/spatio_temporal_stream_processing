Experiment Configuration Parameters
+-----------------------------+-------------------+---------------------------+----------------------------------------------+----------------------------------------------+
| Parameter                   | Category          | Possible Values           | Description                                  | Impact on Experiment                         |
+-----------------------------+-------------------+---------------------------+----------------------------------------------+----------------------------------------------+
| STREAM_MODEL                | Processing Model  | dataflow, microbatch,     | Selects time semantics.                      | Determines event‑time vs processing‑time     |
|                             |                   | actor, log                |                                              | behavior of windows.                         |
+-----------------------------+-------------------+---------------------------+----------------------------------------------+----------------------------------------------+
| WINDOW_STRATEGY             | Windowing         | session, dynamic,         | Selects windowing algorithm.                 | Defines window formation and triggers.       |
|                             |                   | adaptive, multitrigger    |                                              |                                              |
| WINDOW_SIZE                 | Windowing         | Integer (sec)             | Base window size or session gap.             | Controls window boundaries.                  |
| COUNT_THRESHOLD             | Windowing         | Integer                   | Count trigger threshold.                     | Affects multitrigger firing frequency.       |
| PROCESSING_INTERVAL_MS      | Windowing         | Integer (ms)              | Processing‑time trigger interval.            | Controls periodic flushing.                  |
| DENSITY_FACTOR              | Windowing         | Float                     | Adaptive window sensitivity.                 | Controls adaptive window resizing.           |
+-----------------------------+-------------------+---------------------------+----------------------------------------------+----------------------------------------------+
| WATERMARK_OUT_OF_ORDERNESS  | Watermarks        | Integer (sec)             | Allowed lateness for event‑time windows.     | Must be 0 for ideal timestamps; >0 for skew. |
| WATERMARK_MAX_DELAY_MS      | Watermarks        | Integer (ms)              | Fine‑grained lateness override.              | Precise watermark control.                   |
| MICROBATCH_WM_DELAY_MS      | Watermarks        | Integer (ms)              | Watermark delay for microbatch model.        | Defines microbatch granularity.              |
+-----------------------------+-------------------+---------------------------+----------------------------------------------+----------------------------------------------+
| TIMESTAMP_PATTERN           | Timestamp Model   | realtime, skewed, late    | Controls timestamp generation.               | Determines ordering, skew, lateness.         |
| MAX_SKEW_MS                 | Timestamp Model   | Integer (ms)              | Max jitter for skewed timestamps.            | Simulates clock drift.                       |
| MAX_DELAY_MS                | Timestamp Model   | Integer (ms)              | Max lateness for late events.                | Simulates buffering and delayed sensors.     |
+-----------------------------+-------------------+---------------------------+----------------------------------------------+----------------------------------------------+
| EVENT_RATE_PATTERN          | Rate Model        | constant, burst, wave     | Controls event emission pattern.             | Simulates stable, bursty, or periodic load.  |
| EVENT_RATE                  | Rate Model        | Integer (events/sec)      | Base rate for constant pattern.              | Controls throughput.                         |
| BURST_RATE                  | Rate Model        | Integer                   | Rate during burst phase.                     | Simulates high‑load bursts.                  |
| BURST_DURATION_MS           | Rate Model        | Integer (ms)              | Duration of burst phase.                     | Controls burst length.                       |
| PAUSE_MS                    | Rate Model        | Integer (ms)              | Duration of pause phase.                     | Controls idle periods.                       |
| WAVE_MIN                    | Rate Model        | Integer                   | Minimum wave rate.                           | Lower bound of periodic load.                |
| WAVE_MAX                    | Rate Model        | Integer                   | Maximum wave rate.                           | Upper bound of periodic load.                |
| WAVE_PERIOD_MS              | Rate Model        | Integer (ms)              | Duration of wave cycle.                      | Controls oscillation frequency.              |
+-----------------------------+-------------------+---------------------------+----------------------------------------------+----------------------------------------------+
| NUM_VEHICLES                | Key Space         | Integer                   | Number of unique keys.                       | Controls key cardinality and window parallel |
|                             |                   |                           |                                              | ism.                                         |
